# 📄 LFBI LaTeX Paper Template

A clean, ready-to-use LaTeX template for students at **Living Faith Bible Institute (LFBI)** — formatted according to the **Chicago Manual of Style** with a professional cover page layout.

---

## ✨ Features

- Chicago Manual of Style formatting
- Times New Roman, 12 pt font, double spaced
- Properly formatted title (cover) page
- Page numbers, headers, and layout ready to go
- Built-in comments showing how to:
  - Add tables
  - Add figures/images
  - Use footnotes

---

## 🛠 How to Use This Template

### ✅ Option 1: Use with Overleaf (Recommended)
1. Go to [https://overleaf.com](https://overleaf.com)
2. Click **“New Project” → “Upload Project”**
3. Upload `main.tex` (and any images or files you need)
4. Start writing your paper!

### ✅ Option 2: Use Locally
1. Download this repo as a ZIP or clone it: git clone https://github.com/your-username/LFBI-LaTeX-Template.git
2. Open `main.tex` in your LaTeX editor (TeXworks, TeXstudio, VS Code, etc.)
3. Compile to PDF using **PDFLaTeX**
